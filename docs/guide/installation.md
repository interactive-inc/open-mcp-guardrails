# Installation

## Requirements

- [Node.js](https://nodejs.org/) >= 23.6.0 (required for native TypeScript config loading)

## Install

```bash
npm install open-mcp-guardrails
```

Or with Bun:

```bash
bun add open-mcp-guardrails
```
